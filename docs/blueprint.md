# **App Name**: TERRA

## Core Features:

- Interactive Disaster Map: Visualize real-time disaster events, their geographical spread, and historical data on a dynamic map using MapLibre GL JS.
- Real-time Incident Updates: Receive and display live incident alerts, severity updates, and unfolding events using Socket.io for immediate situational awareness.
- AI-Powered Simulation Tool: A tool to simulate various disaster scenarios, analyze potential impacts, and evaluate response strategies based on AI models.
- Disaster Forecasting Module: Utilize predictive analytics and machine learning to forecast disaster trajectories, likely affected areas, and potential resource needs.
- Critical Resource Display: Display locations of critical resources (e.g., emergency services, shelters, supply depots) overlaid on the interactive map.
- Unified Dashboard & Navigation: Provide a centralized dashboard for key metrics and seamless navigation across core application sections like simulation, forecast, and resources via shared header and sidebar.
- User Preference Settings: Allow users to customize display preferences, alert notifications, and map overlays via a dedicated settings interface.

## Style Guidelines:

- Scheme: Dark theme, creating a modern, professional, and focus-oriented environment.
- Primary color: A sophisticated and professional deep blue (#45AFDB) to represent data, intelligence, and calm under pressure.
- Background color: A very dark, desaturated slate-blue (#1E282B) providing a subtle depth that complements the primary and glass effects.
- Accent color: A vibrant yet harmonious indigo (#7D66ED) to highlight interactive elements, calls to action, and critical information.
- Headline and Body font: 'Inter' (sans-serif) for its modern, objective, and highly readable characteristics, suitable for data-dense dashboards.
- Use sleek, minimalist line icons that align with the 'professional dashboard design' and 'glass UI' aesthetic for clarity and sophistication.
- Employ a professional dashboard design with clear hierarchical sections, utilizing a persistent header and sidebar for intuitive navigation and data access.
- Incorporate subtle animations for UI elements, especially hover and click states for 'Glass UI' components, and smooth transitions for data updates or page changes.
